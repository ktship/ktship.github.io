RunPod 설치 가이드

ComfyUI 기본 세팅부터 Qwen / PixelDiT 모델까지

1. ComfyUI 및 매니저 설치 (Linux / RunPod)

① ComfyUI 본체 설치

# 1. ComfyUI 깃허브 저장소 복제 (/workspace 등 원하는 경로에서 실행)
git clone https://github.com/comfyanonymous/ComfyUI.git

# 2. 폴더로 이동
cd ComfyUI

# 3. 필요한 파이썬 패키지 설치
pip install -r requirements.txt

② ComfyUI Manager 설치 (필수)

# 1. 커스텀 노드 폴더로 이동
cd custom_nodes

# 2. 매니저 저장소 복제
git clone https://github.com/ltdrdata/ComfyUI-Manager.git

# 3. 원래 폴더(ComfyUI)로 복귀
cd ..

2. ComfyUI 실행 및 접속 방법

① 서버 실행 (RunPod 터미널)

외부 접속 허용 옵션(--listen)을 붙여 서버를 구동합니다.

cd /workspace/ComfyUI
python main.py --listen 0.0.0.0 --port 8188

② 맥북 SSH 키 수동 등록 및 접속 (현재 방식 유지)

현재 만들어둔 Pod를 그대로 살리면서 맥북 터미널에서 확실하게 접속하고 싶을 때 사용하는 방법입니다.

  1. 맥북 터미널에서 아래 명령어를 입력하여 본인의 '공개키'를 화면에 띄우고, 출력된 텍스트 전체를 복사합니다. (ssh-ed25519 AAA... 로 시작합니다.) 
    cat ~/.ssh/id_ed25519.pub
  2. RunPod 웹 대시보드로 이동하여 해당 Pod의 Connect 버튼을 누르고, Start Web Terminal을 클릭해 브라우저용 까만 터미널 창을 엽니다.
  3. Web Terminal 창에 아래 명령어를 복사해서 붙여넣습니다. (두 번째 줄의 따옴표 "" 사이에 아까 복사한 공개키를 넣으시면 됩니다.) 
    mkdir -p ~/.ssh
echo "여기에_복사한_공개키를_붙여넣으세요" >> ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
  4. 이제 다시 맥북 터미널로 돌아가서 포트 포워딩 접속 명령어를 치시면, 비밀번호를 묻지 않고 스르륵 접속이 될 겁니다! 
    ssh -L 8188:127.0.0.1:8188 root@[IP주소] -p [포트번호] -i ~/.ssh/id_ed25519
  5. 로그인이 완료되어 터미널 창이 켜진 상태에서, 내 PC 인터넷 브라우저 주소창에 http://localhost:8188을 입력하면 ComfyUI 화면이 열립니다.

4. Qwen Image Edit (Rapid-AIO v23)

다운로드

hf download Phr00t/Qwen-Image-Edit-Rapid-AIO \
  --include "v23/Qwen-Rapid-AIO-NSFW-v23.safetensors" \
  --local-dir /workspace/ComfyUI/models/checkpoints

커스텀 노드 설치

ComfyUI Manager에서 Comfyui-QwenEditUtils 검색 후 설치하세요.

워크플로우

워크플로우 열기 (Qwen-Rapid-AIO.json)
VRAM 요구사항
• FP8 버전 기준: 16~20GB
• 안정적으로 사용하려면 20~24GB 이상 권장

5. PixelDiT 설치

다운로드

# 메인 모델
hf download Comfy-Org/PixelDiT \
  --include "pixeldit_1300m_1024px_bf16.safetensors" \
  --local-dir /workspace/ComfyUI/models/diffusion_models

# 텍스트 인코더
hf download Comfy-Org/PixelDiT \
  --include "gemma_2_2b_it_elm_bf16.safetensors" \
  --local-dir /workspace/ComfyUI/models/text_encoders

워크플로우

워크플로우 열기 (PixelDiT)
VRAM 요구사항
• bf16 버전 기준: 10~14GB
• mxfp8 버전 사용 시 8~12GB에서도 동작 가능
