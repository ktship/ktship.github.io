RunPod 설치 가이드

Qwen Image Edit vs PixelDiT • ComfyUI

1. 사전 준비 (Linux / RunPod)

# huggingface-cli 설치 (최초 1회만 실행)
pip install -U huggingface_hub

2. Qwen Image Edit (Rapid-AIO v23)

다운로드

huggingface-cli download Phr00t/Qwen-Image-Edit-Rapid-AIO \
  --include "Qwen-Rapid-AIO-NSFW-v23.safetensors" \
  --local-dir /workspace/ComfyUI/models/checkpoints

커스텀 노드 설치

cd /workspace/ComfyUI/custom_nodes

git clone https://github.com/Comfy-Org/ComfyUI-Manager.git
cd ComfyUI-Manager
pip install -r requirements.txt

ComfyUI를 재시작한 후, ComfyUI Manager에서 TextEncodeQwenImageEditPlus 검색 후 설치하세요.

3. PixelDiT 설치

다운로드

# 메인 모델
huggingface-cli download Comfy-Org/PixelDiT \
  --include "pixeldit_1300m_1024px_bf16.safetensors" \
  --local-dir /workspace/ComfyUI/models/diffusion_models

# 텍스트 인코더
huggingface-cli download Comfy-Org/PixelDiT \
  --include "gemma_2_2b_it_elm_bf16.safetensors" \
  --local-dir /workspace/ComfyUI/models/text_encoders
