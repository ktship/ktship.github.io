Qwen-Image-Edit 2511 및 PixelDiT 원클릭 구축 패키지

1. 원클릭 통합 환경 구축 쉘 스크립트

아래 스크립트 블록 전체를 복사하여 RunPod 터미널에 붙여넣으면 경로 링크 생성, 필수 커스텀 노드 설치, 라이브러리 종속성 해결, Qwen 2511 최신 모델 에코시스템 다운로드 및 경로 정돈까지 한 번에 완료됩니다.

#!/bin/bash

# ==========================================
# 0. 경로 심볼릭 링크 생성 (RunPod 전용 경로 연결)
# ==========================================
echo "=== [0/4] ComfyUI 경로 심볼릭 링크 생성 중... ==="
rm -rf /workspace/ComfyUI
ln -s /workspace/runpod-slim/ComfyUI/ /workspace/ComfyUI


# ==========================================
# 1. 필수 커스텀 노드 패키지 설치
# ==========================================
echo "=== [1/4] 커스텀 노드 다운로드 중... ==="
cd /workspace/ComfyUI/custom_nodes

# Qwen 멀티앵글 카메라 제어 노드 설치
git clone https://github.com/jtydhr88/ComfyUI-qwenmultiangle.git

# Easy-Use 공식 패키지 설치
git clone https://github.com/yolain/ComfyUI-Easy-Use.git

# LayerStyle 전처리 패키지 설치
git clone https://github.com/chflame163/ComfyUI_LayerStyle.git

# 오픈 포즈 에디터
git clone https://github.com/hinablue/ComfyUI_3dPoseEditor.git
            
# ==========================================
# 2. 종속성 라이브러리 패키지 설치
# ==========================================
echo "=== [2/4] 종속성 Python 라이브러리 설치 중... ==="
pip install -r ComfyUI-Easy-Use/requirements.txt
pip install -r ComfyUI_LayerStyle/requirements.txt
pip install -U "huggingface_hub[cli]"


# ==========================================
# 3. 모델 파일 다운로드 (hf download)
# ==========================================
echo "=== [3/4] Hugging Face에서 모델 다운로드 중... ==="

# 1. 메인 모델 (Qwen Image Edit 2511 - FP8 Mixed)
hf download \
  Comfy-Org/Qwen-Image-Edit_ComfyUI \
  split_files/diffusion_models/qwen_image_edit_2511_fp8mixed.safetensors \
  --local-dir /workspace/ComfyUI/models/diffusion_models

# 2. 라이트닝 로라 (4스텝 고속 생성용 - FP32 프리시전)
hf download \
  lightx2v/Qwen-Image-Edit-2511-Lightning \
  Qwen-Image-Edit-2511-Lightning-4steps-V1.0-fp32.safetensors \
  --local-dir /workspace/ComfyUI/models/loras

# 3. 텍스트 인코더 (Qwen 2.5 VL 7B FP8 Scaled)
hf download \
  Comfy-Org/Qwen-Image_ComfyUI \
  split_files/text_encoders/qwen_2.5_vl_7b_fp8_scaled.safetensors \
  --local-dir /workspace/ComfyUI/models/text_encoders

# 4. VAE (Qwen Image VAE)
hf download \
  Comfy-Org/Qwen-Image_ComfyUI \
  split_files/vae/qwen_image_vae.safetensors \
  --local-dir /workspace/ComfyUI/models/vae

# 5. 다각도 뷰 로라 (Multiple Angles LoRA)
hf download \
  fal/Qwen-Image-Edit-2511-Multiple-Angles-LoRA \
  qwen-image-edit-2511-multiple-angles-lora.safetensors \
  --local-dir /workspace/ComfyUI/models/loras


# ==========================================
# 4. 분할 파일(split_files) 경로 정리정돈
# ==========================================
echo "=== [4/4] 하위 폴더에 담긴 모델들 위치 정리 중... ==="
mv /workspace/ComfyUI/models/diffusion_models/split_files/diffusion_models/* /workspace/ComfyUI/models/diffusion_models/ 2>/dev/null
mv /workspace/ComfyUI/models/text_encoders/split_files/text_encoders/* /workspace/ComfyUI/models/text_encoders/ 2>/dev/null
mv /workspace/ComfyUI/models/vae/split_files/vae/* /workspace/ComfyUI/models/vae/ 2>/dev/null

echo "=== 모든 설치가 완벽히 완료되었습니다! ComfyUI 서버를 재시작해 주세요. ==="

2. Qwen Image Edit (Rapid-AIO v23) 레거시 및 PixelDiT 추가 설치

필요에 따라 아래 개별 추가 모델 및 워크플로우를 활용할 수 있습니다.

① Qwen Image Edit (Rapid-AIO v23)

hf download Phr00t/Qwen-Image-Edit-Rapid-AIO   --include "v23/Qwen-Rapid-AIO-NSFW-v23.safetensors"   --local-dir /workspace/ComfyUI/models/checkpoints

※ ComfyUI Manager에서 Comfyui-QwenEditUtils 검색 후 수동 설치가 필요합니다.

워크플로우 열기 (Qwen-Rapid-AIO.json)

② PixelDiT 설치

# 메인 모델 및 텍스트 인코더 다운로드
hf download Comfy-Org/PixelDiT   --include "pixeldit_1300m_1024px_bf16.safetensors"   --local-dir /workspace/ComfyUI/models/diffusion_models

hf download Comfy-Org/PixelDiT   --include "gemma_2_2b_it_elm_bf16.safetensors"   --local-dir /workspace/ComfyUI/models/text_encoders
워크플로우 열기 (PixelDiT)
사양 요구사항 가이드
• Qwen-Rapid-AIO v23: FP8 기준 VRAM 16~20GB (24GB 권장)
• PixelDiT: bf16 기준 VRAM 10~14GB (mxfp8 사용 시 8~12GB 가능)
