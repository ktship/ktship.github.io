1. ComfyUI 및 매니저 설치 (Linux / RunPod)

① ComfyUI 본체 설치

# 1. ComfyUI 깃허브 저장소 복제 (/workspace 등 원하는 경로에서 실행)
git clone https://github.com/comfyanonymous/ComfyUI.git

# 2. 폴더로 이동
cd ComfyUI

# 3. 필요한 파이썬 패키지 설치
pip install -r requirements.txt

② ComfyUI Manager 설치 (필수)

# 1. 커스텀 노드 폴더로 이동
cd custom_nodes

# 2. 매니저 저장소 복제
git clone https://github.com/ltdrdata/ComfyUI-Manager.git

# 3. 원래 폴더(ComfyUI)로 복귀
cd ..

2. 모델 다운로드 사전 준비

# hf 명령어 사용을 위한 허깅페이스 허브 설치
pip install -U huggingface_hub

5. ComfyUI 실행 및 접속 방법

① 실행하기 (터미널)

RunPod 웹 터미널이나 JupyterLab 터미널에서 아래 명령어를 입력하여 ComfyUI를 실행합니다. 이때 외부 접속을 허용하는 --listen 옵션을 반드시 붙여야 합니다.

# 1. ComfyUI 폴더로 이동
cd /workspace/ComfyUI

# 2. 외부 접속 허용(--listen) 및 포트 지정하여 실행
python main.py --listen 0.0.0.0 --port 8188

실행이 정상적으로 되면 터미널 맨 아래쪽에 To see the GUI go to: http://0.0.0.0:8188 주소가 뜨며 대기 상태가 됩니다.

② 접속 방법 1: 포트 개방으로 1초 만에 접속하기 (⭐ 가장 추천)

SSH 터미널 창을 계속 켜두는 번거로움 없이, RunPod 설정에 포트 하나만 추가해주면 대시보드에서 클릭 한 번으로 ComfyUI 웹 창을 띄울 수 있습니다.

  1. RunPod 대시보드에서 현재 Pod의 전원을 잠시 끕니다 (Stop).
  2. Pod의 톱니바퀴 모양 아이콘(Edit Pod 또는 설정)을 클릭합니다.
  3. 중간쯤에 있는 Expose HTTP Ports 칸을 찾습니다.
  4. 기존에 포트 번호가 적혀 있다면 쉼표 뒤에 8188을 추가합니다. (예: 8888,8188 또는 그냥 8188)
  5. 설정을 저장하고 Pod를 다시 켭니다 (Start).
  6. Web Terminal로 들어가서 똑같이 python main.py --listen 0.0.0.0 --port 8188을 실행합니다.
  7. 이제 RunPod 대시보드에서 Connect 버튼을 눌러보세요. HTTP Service [Port 8188] 이라는 파란 버튼이 예쁘게 생겨 있을 겁니다. 이걸 누르면 맥북 터미널을 거칠 필요 없이 바로 ComfyUI 브라우저가 열립니다!

③ 접속 방법 2: SSH 포트 포워딩 이용 (치트키)

네트워크가 완전히 막혀 있어 위 방법으로 접속이 안 될 때 사용하는 가장 확실한 방법입니다.

  1. RunPod 대시보드에서 Connect를 누르고 SSH Command 주소를 복사합니다.
    예: ssh root@64.xx.xx.xx -p 32123
  2. 내 실제 개인 PC의 터미널(Windows는 cmd창, Mac은 터미널)을 엽니다.
  3. 복사한 명령어 중간에 -L 8188:127.0.0.1:8188을 끼워 넣어서 실행하고 연결(로그인)합니다.
    입력 예시: ssh -L 8188:127.0.0.1:8188 root@64.xx.xx.xx -p 32123
  4. 로그인이 완료되면 그 터미널 창을 그대로 켜둔 상태에서, 내 PC 인터넷 브라우저 주소창에 http://localhost:8188을 입력하면 ComfyUI 화면이 열립니다.
