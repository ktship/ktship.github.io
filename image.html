RunPod 설치 가이드

ComfyUI 기본 세팅부터 Qwen / PixelDiT 모델까지

1. ComfyUI 및 매니저 설치 (Linux / RunPod)

① ComfyUI 본체 설치

# 1. ComfyUI 깃허브 저장소 복제 (/workspace 등 원하는 경로에서 실행)
git clone https://github.com/comfyanonymous/ComfyUI.git

# 2. 폴더로 이동
cd ComfyUI

# 3. 필요한 파이썬 패키지 설치
pip install -r requirements.txt

② ComfyUI Manager 설치 (필수)

# 1. 커스텀 노드 폴더로 이동
cd custom_nodes

# 2. 매니저 저장소 복제
git clone https://github.com/ltdrdata/ComfyUI-Manager.git

# 3. 원래 폴더(ComfyUI)로 복귀
cd ..

2. ComfyUI 실행 및 웹 접속 방법

① 서버 실행 (RunPod 터미널)

외부 접속 허용 옵션(--listen)을 붙여 서버를 구동합니다.

cd /workspace/ComfyUI
python main.py --listen 0.0.0.0 --port 8188

② 브라우저 접속 방법 (택 1)

  • 방법 1: RunPod 포트 개방 (가장 추천 ⭐)
    Pod 전원 종료(Stop) → Edit Pod 클릭 → Expose HTTP Ports 칸에 8188 추가 (예: 8888,8188) → 저장 후 Pod 켜기(Start) → 터미널에서 ComfyUI 다시 실행 → 대시보드의 Connect 메뉴에서 HTTP Service [Port 8188] 파란 버튼을 클릭하여 바로 접속합니다.
  • 방법 2: SSH 포트 포워딩 (인증 오류 발생 시 대안)
    본인 개인 PC 터미널을 열고, 터미널 복사 주소 중간에 포트 포워딩 옵션을 추가하여 진입합니다.
    ssh -L 8188:127.0.0.1:8188 root@[IP주소] -p [포트번호]
    개인키(Identity 파일) 지정이 필요한 경우 끝에 -i ~/.ssh/id_ed25519를 붙여 로그인 상태를 유지한 뒤, 브라우저 주소창에 http://localhost:8188을 입력합니다.

4. Qwen Image Edit (Rapid-AIO v23)

다운로드

hf download Phr00t/Qwen-Image-Edit-Rapid-AIO \
  --include "v23/Qwen-Rapid-AIO-NSFW-v23.safetensors" \
  --local-dir /workspace/ComfyUI/models/checkpoints

커스텀 노드 설치

ComfyUI Manager에서 Comfyui-QwenEditUtils 검색 후 설치하세요.

워크플로우

워크플로우 열기 (Qwen-Rapid-AIO.json)
VRAM 요구사항
• FP8 버전 기준: 16~20GB
• 안정적으로 사용하려면 20~24GB 이상 권장

5. PixelDiT 설치

다운로드

# 메인 모델
hf download Comfy-Org/PixelDiT \
  --include "pixeldit_1300m_1024px_bf16.safetensors" \
  --local-dir /workspace/ComfyUI/models/diffusion_models

# 텍스트 인코더
hf download Comfy-Org/PixelDiT \
  --include "gemma_2_2b_it_elm_bf16.safetensors" \
  --local-dir /workspace/ComfyUI/models/text_encoders

워크플로우

워크플로우 열기 (PixelDiT)
VRAM 요구사항
• bf16 버전 기준: 10~14GB
• mxfp8 버전 사용 시 8~12GB에서도 동작 가능
