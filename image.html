RunPod 추천 비교: Qwen Image Edit vs PixelDiT

목적: 이미지 편집 (특히 NSFW 가능 여부 중심)

1. Qwen Image Edit (Rapid-AIO)

워크플로우
첨부된 파일 사용
Qwen-Rapid-AIO.json
NSFW 가능 여부
가능 (Phr00t의 NSFW 버전 사용 시)
Qwen 공식 모델보다는 훨씬 잘 됩니다.
RunPod에서 다운받을 파일
파일경로
메인 모델 (NSFW 추천) Phr00t/Qwen-Image-Edit-Rapid-AIO
→ v19 NSFW 또는 v23 NSFW 버전
커스텀 노드 TextEncodeQwenImageEditPlus
(ComfyUI Manager에서 설치)
VRAM 요구사항
20~24GB 이상 권장 (FP8 버전 사용 시 16~20GB 가능)
주의: 공식 Qwen 모델은 NSFW가 거의 안 되므로, 반드시 Phr00t NSFW 버전을 사용하세요.

2. PixelDiT (Core)

워크플로우
ComfyUI 내장 템플릿 또는 공식 문서 워크플로우 사용
https://docs.comfy.org/tutorials/image/pixeldit/pixeldit
NSFW 가능 여부
Qwen보다는 잘 됩니다.
완전 무검열은 아니지만, Qwen보다는 검열이 약합니다.
RunPod에서 다운받을 파일
파일경로
메인 모델 Comfy-Org/PixelDiT
→ pixeldit_1300m_1024px_bf16.safetensors
텍스트 인코더 Comfy-Org/PixelDiT
→ gemma_2_2b_it_elm_bf16.safetensors
VRAM 요구사항
10~14GB 정도 (mxfp8 버전 사용 시 더 낮아짐)
주의: NSFW는 어느 정도 가능하지만, 매우 노골적인 콘텐츠는 품질이 떨어질 수 있습니다.

최종 추천

목적 추천 모델 이유
NSFW 위주 Qwen Image Edit (Rapid-AIO NSFW) NSFW 성능이 더 좋고, 편집 기능이 강력함
가벼운 사용 + NSFW 가볍게 PixelDiT VRAM 적게 먹고, Qwen보다는 검열이 약함
둘 다 테스트해보고 싶을 때 둘 다 설치 추천 VRAM이 충분하다면 둘 다 써보는 게 좋음
