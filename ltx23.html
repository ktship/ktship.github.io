🐉 RunPod ComfyUI: Sulphur 2 & LTX-2.3 완전체 가이드

본 문서는 고사양 GPU 클라우드인 런포드(RunPod) 환경에서 Sulphur 2 무검열 비디오 및 오디오 생성 모델을 완벽하게 구동하기 위한 마스터 설치 가이드입니다. 터미널 명령어를 복사해 순서대로 붙여넣으세요.

🛠️ 1. 다운로드 스크립트 (Terminal Copy & Paste)

1-1. 메인 체크포인트 (Checkpoint)

순정 bf16 모델(A100/H100 추천, 32G ram)과 FP8 경량화 모델(4090 추천 16G ram) 중 선택하여 다운로드하세요.

cd /workspace/ComfyUI/models/checkpoints/

# 옵션 A (압축 안 한 순정 bf16, 32G 이상)
wget -c -O sulphur_dev_bf16_model.safetensors "https://huggingface.co/vantagewithai/Sulphur-2-Base-Split/resolve/main/model/sulphur_dev_bf16_model.safetensors"

# 옵션 B (FP8 경량화, 16G 이상)
wget -c -O sulphur_dev_model_fp8mixed.safetensors "https://huggingface.co/vantagewithai/Sulphur-2-Base-Split/resolve/main/model/sulphur_dev_model_fp8mixed.safetensors"

1-2. 가속화 로라 (LoRA)

비주얼 디테일을 가다듬고 8스텝 이하의 초고속 생성을 가능하게 해주는 필수 로라입니다.

cd /workspace/ComfyUI/models/loras/
wget -c -O ltx-2.3-22b-distilled-1.1_lora-dynamic_fro09_avg_rank_111_bf16.safetensors "https://huggingface.co/Kijai/LTX2.3_comfy/resolve/main/loras/ltx-2.3-22b-distilled-1.1_lora-dynamic_fro09_avg_rank_111_bf16.safetensors"

1-3. 텍스트 인코더 (Text Encoder)

구글 Gemma 3 12B 기반의 정식 텍스트 인코더 모델입니다.

cd /workspace/ComfyUI/models/text_encoders/
wget -c -O gemma3_ltx_text.safetensors "https://huggingface.co/Pavpif/ltx2-gemma3-text-encoder/resolve/main/model_gemma_3_12B_it_fp8_e4m3fn.safetensors?download=true"

1-4. 멀티모달 VAE (Video & Audio VAE)

영상 픽셀을 제어하는 Video VAE와 비디오 내부 음향을 구현하는 Audio VAE 세트입니다.

cd /workspace/ComfyUI/models/vae/

# 영상(비디오)용 VAE
wget -c -O ltx_video_vae.safetensors "https://huggingface.co/Lightricks/LTX-2/resolve/main/vae/diffusion_pytorch_model.safetensors?download=true"

# 음성(오디오)용 VAE
wget -c -O ltx_audio_vae.safetensors "https://huggingface.co/Lightricks/LTX-2/resolve/main/audio_vae/diffusion_pytorch_model.safetensors?download=true"

1-5. 무검열 프롬프트 인핸서 (Prompt Enhancer LLM)

성인물/고어 연출 시 거절 없이 프롬프트를 풍부한 소설체로 확장해 주는 LLM입니다.

mkdir -p /workspace/ComfyUI/models/LLM
cd /workspace/ComfyUI/models/LLM
wget -O Sulphur_Prompt_Enhancer.gguf https://huggingface.co/Seregil13th/Sulphur-2-base/resolve/main/promptenhancer/Sulphur_Prompt_Enhancer.gguf

1-6. 모션 컨트롤넷 (ControlNet 2종)

포즈(OpenPose)와 입체감(Depth)을 제어하여 신체 꼬임과 물리 오류를 방지합니다.

cd /workspace/ComfyUI/models/controlnet/

# 포즈 제어용 (OpenPose)
wget -O ltx2_3_controlnet_openpose.safetensors https://huggingface.co/Lightricks/LTX-2-3/resolve/main/controlnet/openpose.safetensors

# 깊이 제어용 (Depth)
wget -O ltx2_3_controlnet_depth.safetensors https://huggingface.co/Lightricks/LTX-2-3/resolve/main/controlnet/depth.safetensors

1-7. 공간 업스케일러 (Spatial Upscaler)

저해상도 비디오를 화질 저하 없이 1080p, 4K 등 영화급 화질로 벼려내는 전용 업스케일러입니다.

mkdir -p /workspace/ComfyUI/models/upscale_models
cd /workspace/ComfyUI/models/upscale_models
2배 1.1버젼
wget -c -O ltx_upscaler_x2_v1.1.safetensors "https://huggingface.co/Lightricks/LTX-2.3/resolve/main/ltx-2.3-spatial-upscaler-x2-1.1.safetensors?download=true"
1.5배 
wget -c -O ltx_upscaler_x1.5.safetensors "https://huggingface.co/Lightricks/LTX-2.3/resolve/main/ltx-2.3-spatial-upscaler-x1.5-1.0.safetensors?download=true"

latent upsampler(이건 옵션)
wget -c -O ltx_latent_upsampler.safetensors "https://huggingface.co/Lightricks/LTX-2/resolve/main/latent_upsampler/diffusion_pytorch_model.safetensors?download=true"

🚀 3. 구동 및 연결 방법

  1. 모든 파일 다운로드가 완료되면 ComfyUI 웹 브라우저 우측 메뉴에서 [Refresh] 버튼을 누릅니다.
  2. Sulphur 2 / LTX-2.3 전용 오디오·비디오 결합 워크플로우를 가져옵니다.
  3. 각 로드 노드(Load Checkpoint, Load VAE 등)에 파일명이 제대로 매칭되어 있는지 확인합니다.
  4. 프롬프트 창에 원하는 성인물/시네마틱 연출을 입력하고 [Queue Prompt]를 눌러 무검열 영상 생성을 시작합니다.
